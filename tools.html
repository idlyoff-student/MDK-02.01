Инструменты Kali Linux

Мощный арсенал для тестирования безопасности

Nmap

Сканер сети

Мощный инструмент для сканирования сети и аудита безопасности. Позволяет обнаруживать хосты и сервисы в компьютерной сети.

Сканирование портов Определение ОС Скрипты NSE
nmap -sV -p 1-1000 192.168.1.1 --- Пример использования

Wireshark

Анализатор пакетов

Анализатор сетевого трафика для захвата и интерактивного просмотра содержимого сетевых пакетов.

Анализ протоколов Фильтрация пакетов Декодирование

Metasploit

Фреймворк тестирования

Платформа для разработки, тестирования и использования эксплойтов. Содержит базу данных проверенных эксплойтов.

База эксплойтов Модульная структура Payload генератор

Aircrack-ng

Аудит Wi-Fi сетей

Набор инструментов для оценки безопасности Wi-Fi сетей, включая анализ пакетов и тестирование ключей.

WEP/WPA взлом Мониторинг сетей Инъекция пакетов

John the Ripper

Взлом паролей

Инструмент для подбора паролей, поддерживающий различные алгоритмы хеширования.

Множество форматов GPU ускорение Гибкие правила

Burp Suite

Тестирование веб-приложений

Интегрированная платформа для проведения аудита безопасности веб-приложений.
